Deprecated Tags

Deprecated tags are tags that are no longer supported by HTML.

Tag Description Alternate
acronym Allows authors to clearly indicate a sequence of characters that compose an acronym or abbreviation for a word. abbr
listing It was intended as a way to render HTML code on a page. pre
dir Container for a directory of files and/or folders, potentially with styles and icons applied by the user agent. ul
image An ancient and poorly supported precursor to the image element. img
strike Places a strikethrough (horizontal line) over text. del
u It is used to create underlined text. css...text-decoration
spacer This tag is used to specify a whitespace br
center It is used to set the alignment of the text. css...text-align
